# Hardware Specifications

## 1. Motor and Power Transmission

| Specification | Value |
| --- | --- |
| Motor Type | MY1016 Brushed DC Motor |
| Rated Power | 250W |
| Voltage | 24V |
| Sprocket Size | 88-tooth sprocket |
| Gear Ratio | 1:8 |
| Mechanical Advantage | 8× |
| Torque Multiplication | Base 0.8 Nm → 6.4 Nm effective |
| RPM Characteristic | High RPM motor suitable for bicycle speeds |

The motor selection was critical. A high-RPM, low-torque motor was deliberately chosen because the gearing system (sprocket-freewheel arrangement) provides the necessary mechanical advantage. This approach offers several advantages over direct-drive systems:

- Better efficiency at low speeds
- Reduced mechanical stress on components
- Compatibility with standard bicycle chain systems
- Cost reduction through use of proven automotive components

## 2. Power Storage

| Specification | Value |
| --- | --- |
| Battery Type | Sealed Lead-Acid (SLA) – UPS Type |
| Configuration | Two batteries in series |
| Voltage per Battery | 12V |
| Total System Voltage | 24V |
| Capacity per Battery | ~7 Ah |
| Source | Recycled from discarded computer UPS systems |
| Advantages | Maintenance-free, readily available, cost-effective |

**Note**: The initial plan to use lithium-ion batteries was abandoned due to cost considerations. Sealed lead-acid batteries from old UPS systems provide a sustainable alternative at minimal cost. While heavier than lithium batteries, they are proven, safe, and widely available from e-waste sources.

## 6. Electrical Protection

All electrical components are housed in a waterproof acrylic protective enclosure with rubber sealing. The enclosure is mounted to the bicycle frame and protects the controller, switching mechanism, and power distribution components from:

- Water damage from rain or washing
- Dust and debris accumulation
- Mechanical impact and vibration
- UV degradation from prolonged sunlight exposure
